# fiap-mba-microservices-exercicio-1

## Inicialização projeto

```sh
mvn sprint-boot:run
```

## Pedidos

#### Listar por id

curl -X GET \
  http://localhost:8080/orders/1 \
  -H 'cache-control: no-cache' \
  -H 'content-type: application/json' \
  -H 'postman-token: 2d4256c0-5a4d-5ec3-569d-68e36e42fbf4' \
  -d '{}'
